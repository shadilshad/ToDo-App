# 📝 To-Do List API (Java Spring Boot)

A simple backend To-Do List application built using **Java**, **Spring Boot**, and **MySQL**.  
This project provides RESTful APIs to create, retrieve, and delete tasks while storing them in a MySQL database.

It demonstrates core backend development concepts such as **REST API design, database integration, and CRUD operations**.

---

## 🚀 Features

- Add a new task
- View all tasks
- Delete a task
- Store tasks in a MySQL database
- Simple REST API structure

---

## 🛠 Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Postman (for API testing)
